Layout

Frågor:

  • Då man skapar en layout med grid används bl.a. CSS-egenskaperna grid-template-columns, grid-template-rows och grid-template-areas. Beskriv vad dessa används till.
    • Grid-template-columns används till att skapa och specifiera bredden på kolumnerna.(Exempel 1)
    • Grid-template-rows används för att skapa och specifiera höjden på raderna.(Exempel 1)
    • Grid-template-areas används för att specifiera grid-mallen genom att referera till angivna namn på specifika område.(Exempel 1, Exempel 2)
  • Vad är en media query och vad används den till?
    • Med hjälp av Media queries gör vi förändringar i koden så att anpassa layouten till specifik skärmstorlek och på så sätt gör webbsidan responsive.(Exempel 3)
